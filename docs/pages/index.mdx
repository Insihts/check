# Insihts Documentation

**Welcome to Insihts!** This is your go-to guide for mastering the Insihts platform—an AI-driven solution that combines analytics, collaboration, and workflows to help your team achieve success.

---

## What is Insihts?

Insihts is an all-in-one platform for businesses looking to track user behavior, optimize workflows, and make data-driven decisions. Whether you're a marketer, product manager, or part of a customer success team, Insihts provides the tools you need to succeed.

### Key Features

  <Cards.Card arrow title="Workflow Creation" href="/features/workflows">

- **Deep User Behavior Analytics**: Understand how users interact with your product or campaigns.
- **AI-Driven Planning**: Automatically generate OKRs, tasks, and strategic documents.
- **Workflows**: Create onboarding flows, product tours, and multi-step user journeys.
- **Collaboration Tools**: Shared dashboards, whiteboards, and Kanban boards for alignment.
- **Experimentation**: Run A/B tests and optimize user experiences based on actionable insights.
